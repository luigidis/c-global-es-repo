# Istruzioni per l'Uso

Di seguito le istruzioni per usare correttamente il programmino in C che permette di ripassare alcuni dei più semplici algoritmi di ordinamento, ognuno implementato diversamente sulla base di diversi possibili scenari nel compito di programmazione 1.

### Utilizzo
```bash
./a.out <sort> <tipologia>

```

### 1. Scelta del parametro `<sort>`

Per scegliere quale algoritmo di ordinamento utilizzare scrivere letteralmente il nome abbreviato dell'algoritmo desiderato al posto del parametro `<sort>` nel seguente modo:

1. `bubble`
2. `insertion`
3. `selection`

> **Nota:** Se l'input inserito dall'utente non sarà conforme a nessuno dei casi previsti, il programma terminerà immediatamente restituendo un errore.

### 2. Scelta del parametro `<tipologia>`

Per scegliere quale implementazione dell'algoritmo desiderato visualizzare inserire al posto del parametro `<tipologia>` uno dei seguenti numeri:

1. Per l'ordinamento di un semplice array monodimensionale.
2. Per l'ordinamento di una matrice di double.
3. Per l'ordinamento di una matrice di puntatori a double.

> **Nota:** Se l'input inserito dall'utente non sarà conforme a nessuno dei casi previsti, il programma terminerà immediatamente restituendo un errore.
